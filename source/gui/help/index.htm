Introduction

The whole idea of Developer Playground is simple: Allow hooking of arbitrary processes and tracing of every call to any DLL, without the overhead of a debugger. We feel that this utility is a great help in tracing program flow without access to neither source code nor debug symbols, and without having to resort to assembly-level debugging, and we hope you'll find it useful as well.
